# NexCartAI

NexCartAI is a frontend-only hackathon prototype for a local marketplace. Open index.html in your browser to explore the experience.

## Run locally

- Open [index.html](index.html) in a modern browser.
- The project uses LocalStorage and sample JSON files so it works without a backend.

## Included pages

- Landing page
- Login and signup
- Buyer home, browse, wishlist, orders, profile, settings
- Seller dashboard, profile, reviews
- Product details and supporting info pages
